What is a link?

In a causal set, x < y is a link if no element lies between them:
x < y ,   ¬∃ z :   x < z < y
Irreducible causal bonds — every relation is composed from them.
They carry the scalar-field propagator (Johnston 2008) and generate the Newtonian potential.

The valency diverges

In a Poisson sprinkling, the expected valency is infinite
The past light cone has infinite volume within any proper-time shell — arbitrarily remote elements can be linked.
